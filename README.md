# Mashnu AI

Enterprise AI Automation & Intelligent AI Employees — marketing site with an agent-compiler playground, execution simulator, lead qualifier, and a personal AI assistant.

No external API, no API keys, no billing required — every interactive demo runs on deterministic sample logic built into the server.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

The server listens on http://localhost:3000.

## Build for Production

```sh
npm run build
npm start
```
